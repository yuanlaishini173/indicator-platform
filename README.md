# 月度指标自动整理平台

这是一个先行原型，用于验证“月度表格导入、指标自动整理、完成情况查询、全市/各区汇总、报告生成”的主流程。当前版本内置模拟数据，后续拿到真实样例表后，只需要重点调整 `app/main.py` 中的字段识别和清洗规则。

## 启动

```powershell
cd C:\Users\lvliq\Documents\Codex\2026-05-30\new-chat\outputs\indicator-platform
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8020
```

浏览器打开：

```text
http://127.0.0.1:8020
```

## 已完成

- Excel/CSV 上传入口
- 字段自动识别框架：区县/单位、指标名称、分类、目标值、完成值、完成率、单位、备注
- 指标完成率计算：默认高值更好，污染、投诉、能耗等指标自动按低值更好处理
- 完成状态判断：已完成、接近完成、未完成、待判断
- 全市汇总：按指标聚合目标值和完成值
- 各区汇总：完成数、预警数、未完成数、平均完成率
- 实时查询：月份、区域、类别、状态、关键词
- Markdown 报告生成与下载
- SQLite 本地数据存储

## 后续接样例表时重点调整

- 表头定位：内网导出表如果有多行标题、合并单元格，需要补专用清洗规则
- 指标口径：不同指标是否有不同完成阈值、是否要加权
- 全市汇总：有些指标适合求和，有些适合平均或取最新值
- 报告模板：可继续生成 Word/PDF，并加入原因分析、责任单位、整改期限
